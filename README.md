# Albert-bot
Um bot do discord criado para um grupo de amigos

Para inicializar o bot sera necesario o seu token de bot, para fazer isso crie um arquivo com o nome ".env" e coloque esse
conteudo dentro, não use aspas!

DISCORD_TOKEN=SEU_TOKEN_AQUI


Para usar /random sera necessario colocar imagens (gifs ou videos não tem suporte) nas pastas com suas respectivas raridades


# Inicializando
terminal, um commando por vez:

cd "path da pasta do bot aqui" exemplo no linux: cd /home/conductor/Documents/alber/

python main.py
